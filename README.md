# LL 命令行工具（llcmd）

一个面向 Windows 的个人命令行工具集合：
- **电源管理**：倒计时关机、空闲关机监听、取消/查看状态；
- **守护模式**：全屏守护/监控界面（用于观察进程/系统/事件流）；
- **系统工具与快捷操作**：常用系统界面一键打开、网络修复、剪贴板等；
- **快捷启动**：为常用应用注册短命令快速启动。

> 目标：把日常“懒得点”的操作变成短命令，减少鼠标操作，保持输出简洁。

---

## 功能概览

### 1) 交互模式
直接运行程序不带参数会进入交互模式：

- 默认启动空闲关机监听（可通过命令取消）
- 按提示输入任意命令执行

### 2) 电源管理
- `sd <时间>`：倒计时关机（如 `sd 30m` / `sd 2h`）
- `idle <时间>`：空闲达到阈值后关机（如 `idle 1h`）
- `st`：查看任务状态
- `c`：取消当前任务
- `abort`：中止系统关机（相当于 `shutdown /a`）

### 3) 守护模式
- `gd`：进入/退出守护模式

守护模式用于：
- 展示进程/系统关键指标
- 事件流（采样/触发类事件）

> 说明：项目会尽量避免“刷屏式提示”，保持界面干净。

### 4) 便捷系统操作（短命令）
这些命令用于普通用户日常快速打开系统功能：

- `task`：任务管理器
- `dev`：设备管理器
- `ctrl`：控制面板
- `set`：系统设置
- `netset`：网络设置
- `sound`：声音设置
- `disp`：显示设置
- `store`：存储管理（存储/存储感知）
- `desk`：打开桌面目录
- `tmp`：打开临时目录
- `recycle`：打开回收站
- `snip`：截图工具
- `dnsflush`：刷新 DNS 缓存
- `netfix`：网络修复（会执行常见网络重置命令，部分需要管理员权限）

### 5) 其他常用工具
- `ps [关键词]`：列进程（可过滤）
- `kill <pid|name>`：结束进程
- `ip`：查看网络信息
- `port <端口>`：端口监听检测（简版）
- `curl <url>`：HTTP GET（简版）
- `dns <域名>`：DNS 解析
- `find <关键字> [目录]`：按文件名递归查找
- `watch <目录>`：监听目录变化
- `clip [文本]`：读取/写入剪贴板
- `path [filter]`：查看 PATH
- `env [filter]`：查看环境变量
- `clean temp`：清理临时目录（简版）

## 安装与运行

### 运行
- Windows 上直接运行编译产物即可。

### 构建
使用 .NET SDK：

- 目标框架：`.NET 10`
- 建议在 Windows x64 环境编译

## 项目定位

- 不追求复杂参数体系
- 更强调短命令、中文输出、少提示、能直接解决日常问题
