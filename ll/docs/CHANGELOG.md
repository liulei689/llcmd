- 总运行时长跟踪，每秒更新。

- 统一config.json读写逻辑，使用ConfigManager。

- 修复长输入显示问题，显示末尾部分。
- 优化Hex查看输出格式为标准hexdump，多行全部显示。
- 改进长输入显示，长输入显示全部，光标设到末尾。
- 修改Hex查看索引为十进制偏移。
- 修改Hex查看索引为行号递增。
