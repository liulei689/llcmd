# CHANGELOG.md - LL CLI TOOL

## [Unreleased]
- 总运行时长跟踪，每秒更新。
- 守护界面显示总运行时长。
- 添加启动最小化配置参数。
- 统一config.json读写逻辑，使用ConfigManager。
- 添加粘贴文本功能，使用paste命令。
- 添加全局热键 Ctrl+Shift+V 输入预定文本。
- 增强剪贴板监控功能，支持20+种文本快捷插入模板（开发、文档、日常工作等）。
- 添加数学表达式自动计算功能，支持小数输出，使用正则表达式识别运算符。
- 添加图片分析功能，获取像素数量和颜色分布，包含文件信息、EXIF、亮度直方图、对比度、饱和度等。
- 添加二进制文件读取功能，支持十六进制和二进制字符串输出，自动打开记事本查看。
